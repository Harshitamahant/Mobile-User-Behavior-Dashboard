# 📱 User Behavior Analytics Dashboard

> An interactive Power BI dashboard designed to analyze mobile usage patterns, engagement trends, battery performance, and user segmentation through data-driven insights.

---

## 🚀 Project Overview

This project focuses on understanding how users interact with mobile devices by analyzing:

- 📲 Application usage behavior
- ⏱️ Screen-on activity
- 🔋 Battery consumption
- 🌐 Data usage trends
- 👥 User demographics
- 📱 Device & Operating System performance

The dashboard transforms raw mobile usage data into actionable business insights using **Power BI**, **DAX**, and **Power Query**.

# 📊 Key Business Insights

### 🔹 User Engagement Trends
Users demonstrate high daily engagement, with average app usage approaching 5 hours per day and screen-on time exceeding 5.7 hours.

### 🔹 Demographic Analysis
The user base is broadly distributed across age groups with balanced gender representation, indicating diverse usage behavior.

### 🔹 Operating System Comparison
iOS users exhibit slightly higher app engagement levels, while Android users maintain comparable battery consumption patterns.

### 🔹 Device-Level Consumption
Premium smartphone models such as iPhone 12 and Xiaomi Mi 11 show significantly higher data consumption and user interaction levels.

### 🔹 Power User Identification
A substantial portion of users fall into high-engagement categories, highlighting opportunities for premium targeting and retention strategies.

### 🔹 Battery Usage Patterns
Heavy app usage strongly correlates with increased battery drain, emphasizing the importance of performance optimization.

### 🔹 Behavioral Correlation
A positive relationship exists between app usage, screen-on time, and data consumption, reflecting predictable user behavior patterns.
