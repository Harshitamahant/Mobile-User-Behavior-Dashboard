# 📊 User Behaviour Dashboard (Power BI)

## 🔹 Overview
This project analyzes mobile device usage patterns, user engagement, data consumption, and battery performance using Power BI.

---

## 🔹 Features
- Interactive Power BI dashboard
- User behavior segmentation
- Device and OS comparison
- Power user identification
- High data consumption analysis
- Battery usage insights
- Dynamic slicers and filtering

---

## 🔹 Tools Used
- Power BI
- DAX
- Power Query
- Data Visualization

---

## 🔹 Key Insights
- iOS users show slightly higher engagement levels
- Power users form a major portion of the dataset
- Premium devices consume higher mobile data
- High engagement directly impacts battery consumption
- Strong relationship observed between app usage and screen time
